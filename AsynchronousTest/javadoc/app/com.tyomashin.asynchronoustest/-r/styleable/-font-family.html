Attributes that can be used with a FontFamily.

Includes the following attributes:

AttributeDescription com.tyomashin.asynchronoustest:fontProviderAuthorityThe authority of the Font Provider to be used for the request. com.tyomashin.asynchronoustest:fontProviderCertsThe sets of hashes for the certificates the provider should be signed with. com.tyomashin.asynchronoustest:fontProviderFetchStrategyThe strategy to be used when fetching font data from a font provider in XML layouts. com.tyomashin.asynchronoustest:fontProviderFetchTimeoutThe length of the timeout during fetching. com.tyomashin.asynchronoustest:fontProviderPackageThe package for the Font Provider to be used for the request. com.tyomashin.asynchronoustest:fontProviderQueryThe query to be sent over to the provider.
