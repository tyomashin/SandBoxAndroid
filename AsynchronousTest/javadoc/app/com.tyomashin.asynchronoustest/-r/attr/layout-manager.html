static val layoutManager: Int
static val layoutManager: Int

Class name of the Layout Manager to be used.

The class must extandroidx.recyclerview.widget.RecyclerViewView$LayoutManager and have either a default constructor or constructor with the signature (android.content.Context, android.util.AttributeSet, int, int). If the name starts with a '.', application package is prefixed. Else, if the name contains a '.', the classname is assumed to be a full class name. Else, the recycler view package naandroidx.appcompat.widgetdget) is prefixed.

May be a string value, using '\\;' to escape characters such as '\\n' or '\\uxxxx' for a unicode character;
